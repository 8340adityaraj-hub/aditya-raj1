# 📘 Interactive HTML & CSS Reference Guide

This project is an **interactive HTML & CSS learning website** built using **pure HTML, CSS, and JavaScript**.  
It is designed to help beginners and intermediate learners understand web development concepts with examples, live previews, and quizzes.

---

## ✨ Features

- 📚 Beginner to Advanced HTML & CSS reference
- 🌓 Dark Mode / Light Mode toggle
- 🔽 Expand & Collapse sections
- 📋 Copy code button for examples
- 🧪 Live HTML code editor with real-time preview
- 🧠 Quiz section (Basic, Intermediate, Advanced)
- 📱 Fully responsive design
- ⚡ No frameworks used (Pure HTML, CSS & JavaScript)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
