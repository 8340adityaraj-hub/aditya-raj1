🌐 Interactive HTML Reference Guide

💡 Tip: Click on any tag/property in the bullet lists (like <html>, display) to see its definition. Double-click on tags in code examples.

1️⃣ Beginner Tags
  • <!doctype html>, <html>, <head>, <body>
  • <title>, <meta>, <link> (styles)
  • <h1> to <h6>, <p>, <br>
  • <strong>, <em>, <small>
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Page Title</title>
</head>
<body>
  <h1>Welcome</h1>
  <p>This is a paragraph.<br>New line here.</p>
</body>
</html>
2️⃣ Intermediate: Lists, Links & Images
  • <ul>, <ol>, <li>
  • <a>, <img>, <picture>, <figure>, <figcaption>
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
<a href="https://example.com" target="_blank" rel="noopener">Visit example</a>

<figure>
  <img src="logo.png" alt="Logo" width="200">
  <figcaption>Site logo</figcaption>
</figure>

<picture>
  <source srcset="image.webp" type="image/webp">
  <img src="image.jpg" alt="Fallback image">
</picture>
3️⃣ Intermediate: Containers & Layout
  • <div>, <span>, <hr>, <pre>
  • <nav>, <main>, <section>, <aside>, <footer>
<nav>
  <ul><li>Home</li><li>About</li></ul>
</nav>

<main>
  <section>Main Section</section>
  <aside>Sidebar content</aside>
</main>

<hr>

<pre>
  Line 1
  Line 2
</pre>
4️⃣ Intermediate: Forms & Inputs
  • <form>, <input>, <label>, <textarea>, <select>, <button>
<form action="#" method="post">
  <label for="name">Name:</label>
  <input id="name" name="name" type="text" placeholder="Name">

  <label for="msg">Message:</label>
  <textarea id="msg" name="message">Your message</textarea>

  <label for="topic">Topic:</label>
  <select id="topic" name="topic">
    <option value="general">General</option>
    <option value="feedback">Feedback</option>
  </select>

  <button type="submit">Submit</button>
</form>
5️⃣ Advanced: Tables & Semantic Tags
  • <table>, <caption>, <thead>, <tbody>, <tr>, <td>, <th>
  • <header>, <footer>, <section>, <article>, <nav>
<table border="1">
  <caption>User list</caption>
  <thead>
    <tr><th>Name</th><th>Age</th></tr>
  </thead>
  <tbody>
    <tr><td>Aman</td><td>19</td></tr>
  </tbody>
</table>

<header>Site Header</header>
<section>Main Content</section>
<article>Article content</article>
<footer>Footer Info</footer>
6️⃣ Advanced: Media & Canvas
  • <audio>, <video>, <picture>, <iframe>, <canvas>
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

<picture>
  <source srcset="hero.webp" type="image/webp">
  <img src="hero.jpg" alt="Hero image">
</picture>

<canvas id="c" width="300" height="150">Fallback content for canvas</canvas>

<iframe src="https://example.com" width="300" height="200"></iframe>
7️⃣ CSS Beginner: Basic Styling
  • color, background-color, font-size, font-weight
  • text-align, text-decoration, margin, padding
  • <style> tag and inline style attribute
<style>
  body { font-family: Arial, sans-serif; background-color: #f0f0f0; }
  h1 { color: #333; text-align: center; }
  p { color: #666; font-size: 16px; line-height: 1.6; }
  .highlight { background-color: yellow; padding: 5px; }
</style>

<h1>Hello World</h1>
<p>This is a <span class="highlight">styled</span> paragraph.</p>
8️⃣ CSS Intermediate: Selectors & Box Model
  • Element, class, ID selectors; attribute selectors
  • <border>, border-radius, box-shadow
  • Box model: width, height, margin, padding, border
<style>
  * { box-sizing: border-box; }
  #header { background: #2c3e50; color: white; padding: 20px; }
  .button { background: #3498db; color: white; padding: 10px 20px; 
           border: none; border-radius: 5px; cursor: pointer; }
  input[type="text"] { border: 1px solid #ccc; padding: 8px; }
</style>

<div id="header">
  <button class="button">Click me</button>
  <input type="text" placeholder="Enter text">
</div>
9️⃣ CSS Intermediate: Flexbox & Grid Basics
  • display: flex, justify-content, align-items
  • display: grid, grid-template-columns, gap
  • Responsive layouts with flex-wrap and media queries
<style>
  .flex-container { display: flex; justify-content: space-between; gap: 10px; }
  .flex-item { flex: 1; background: #ecf0f1; padding: 15px; }
  
  .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 15px; }
  .grid-item { background: #3498db; color: white; padding: 20px; text-align: center; }
</style>

<div class="flex-container">
  <div class="flex-item">Flex 1</div>
  <div class="flex-item">Flex 2</div>
</div>

<div class="grid-container">
  <div class="grid-item">Grid 1</div>
  <div class="grid-item">Grid 2</div>
  <div class="grid-item">Grid 3</div>
</div>
🔟 CSS Intermediate: Positioning & Display
  • position: static, relative, absolute, fixed
  • z-index, top, left, right, bottom
  • display: block, inline, inline-block, none
<style>
  .container { position: relative; height: 200px; background: #f9f9f9; }
  .absolute-box { position: absolute; top: 20px; left: 20px; 
                 background: #e74c3c; color: white; padding: 10px; }
  .fixed-nav { position: fixed; top: 0; left: 0; right: 0; 
              background: #2c3e50; z-index: 100; }
  .inline-link { display: inline-block; margin: 5px; }
</style>

<div class="container">
  <div class="absolute-box">Absolute Positioned</div>
</div>
<a href="#" class="inline-link">Link 1</a>
<a href="#" class="inline-link">Link 2</a>
1️⃣1️⃣ CSS Advanced: Transforms & Transitions
  • transform: translate, scale, rotate, skew
  • transition, animation, @keyframes
  • Hover effects and smooth animations
<style>
  .box { width: 100px; height: 100px; background: #3498db; 
         transition: all 0.3s ease; cursor: pointer; }
  .box:hover { transform: scale(1.2) rotate(10deg); background: #e74c3c; }
  
  @keyframes slide { from { transform: translateX(0); } to { transform: translateX(100px); } }
  .slide-box { animation: slide 2s infinite alternate; background: #2ecc71; }
</style>

<div class="box"></div>
<div class="slide-box"></div>
1️⃣2️⃣ CSS Advanced: Gradients, Filters & Effects
  • background: linear-gradient, radial-gradient
  • filter: blur, brightness, contrast, grayscale, shadow
  • Advanced text effects and pseudo-elements ::before, ::after
<style>
  .gradient-box { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); 
                 color: white; padding: 40px; border-radius: 10px; }
  .filtered-img { filter: blur(5px) brightness(1.1); transition: filter 0.3s; }
  .filtered-img:hover { filter: blur(0) brightness(1); }
  
  .pseudo::before { content: "→ "; color: #e74c3c; font-weight: bold; }
</style>

<div class="gradient-box">Gradient Background</div>
<img src="image.jpg" class="filtered-img" alt="Filtered image">
<p class="pseudo">Bullet text</p>
